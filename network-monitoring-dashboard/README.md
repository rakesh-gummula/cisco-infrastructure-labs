# Network Monitoring Dashboard
A real-time network monitoring dashboard built using React, Node.js, and WebSockets.

## Features
- Real-time device monitoring
- Latency tracking
- Packet loss detection
- Live alerts system
- Device inventory management
- Interactive charts and analytics
- Responsive modern UI

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Recharts
- Framer Motion

### Backend
- Node.js
- Express.js
- Socket.IO

### Monitoring Tools
- Ping
- SNMP
- Prometheus (optional)

## Installation

### Clone Repository

```bash
git clone https://github.com/rakesh-gummula/network-monitoring-dashboard.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm start
```
